Cranky Toucan Animated Short

For my senior project, I wanted to explore tool development in Maya in order to learn the basics of working with an artist and developing custom tools to ease development of a project. Below is a test render of the almost finished product.



In a previous art class, I had developed this 2D "Cranky Toucan" character for a logo (below), and was excited to explore Maya from both an artist's perspective as well as a software engineer's. Thus, I developed the idea for an animated short that revolved around this toucan character being awoken by the scent of coffee. Rather than develop an entire story, I just wanted to capture a single scene that would allow me to learn about many different aspects of animation, like modeling, rigging, animated models, and more.



For the Aroma character, I wanted to manually develop this character in Maya, tweaking all the attributes as I went along to both gain an understanding of Maya fluids and particles, as well as to gain an appreciation of custom tool development later on. Once I had designed the Aroma character, I then used python scripts to create a tool window that would do all the heavy lifting for me. This tool would spawn the Aroma with it's essential attributes already set, and allowed the user to then toggle what I deemed the most important attributes as well as insert keyframes or delete the character. My goal for this was to simulate an animation environment where I, as a software engineer, was tasked to create this tool to assist the artists in bringing the character into multiple scenes without having to rebuild the character each time by hand and also to reduce human error. Below is an image of the window for the tool.

For the Toucan character, I modeled it in Maya and textured it in Substance Painter, and I used Mixamo to rig it (Mixamo rig/animation example below). This was my first exposure to working with animations in Maya and I am very excited to learn more about all of the software that went into this project.



I learned a lot developing this project as my first exposure to scripting and modelling in Maya and I am very excited to continue learning about this software.

Jillian Empey
